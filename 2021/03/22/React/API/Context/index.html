API -- React.createContext

Context

参考文章[https://zh-hans.reactjs.org/docs/context.html#gatsby-focus-wrapper]

Context 提供了一个无需为每层组件手动添加props，就能再组件树间进行数据传递的方法。
一般父组件向子组件传值，使用props的方式，但是，如果组件的层级是下边这样的。

Fourth组件需要使用First组件里的一个属性，但是，Second Third 是不需要这个属性的，但是，
他们依旧需要接收这个属性，并向下传递。

Fourth 组件需要的属性每增加或者是减少一个，都得在这个组件树里全部改动一番。

Context 就可以解决这个问题，但是，Context的应用场景是 不同层级的组件都需要访问一样的数据，比如Redux 做的那样，

如果只是想简单的避免属性的层层传递，还有一些其他的方式，参考[../技巧/组件组合.md]
