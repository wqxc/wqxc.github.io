一些概念

横切关注点

部分关注点「横切」程序代码中的数个模块，即在多个模块中都有出现，它们即被称作「横切关注点（Cross-cutting concerns, Horizontal concerns）」。

简单的来说，就是我们的业务逻辑分散在一个组件的不同生命周期中。

比如：在componentDidMount 里 注册监听函数
早 ComponentWilUnMount 里取消注册监听
类似这种横跨多个生命周期函数的逻辑。叫做横切关注点

最早的时候，使用mixins 或者是高阶组件来解决这个问题。
后来：
vue3中出现了setUp 来解决这个问题
React16之后出现了React-Hook 来解决这个问题

派生state

派生的意思是说，一个事物是从另外一个事物的发展中分化出的。
派生state是说，state不是我们在constructor中定义的state，而是使用props派生的。

比如：

state = { email: this.props.email };

state email 派生自 props。
